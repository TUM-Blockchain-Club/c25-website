# Sponsor Logos

This directory contains logo files for all sponsors organized by sponsorship tier.

## Logo Requirements

- Format: PNG or SVG preferred
- Dimensions: Recommended minimum 200x200px for PNG files
- Background: Transparent preferred for PNG files
- Quality: High resolution suitable for web display

## Adding New Sponsors

1. Add the logo file to this directory
2. Update the appropriate sponsors array in `sections/Sponsors.tsx`
3. Follow the existing naming convention: `company_name_logo.png` or `company_name_logo.svg`
